About the problem at hand

Remembering numerous passwords from various sites can be difficult for a user. So to provide some flexibility we can provide users a graphical password authentication system where instead of creating a password a user has to select graphical objects in a it as their particular order to keep password.

About The App

We have provided a webpage with some minigames which are walled behind an authentication system that uses graphical image after making user choose their favourite categories for authentication. The code for authentication is easily extractable to be reused in other webpages.

About the Authentication

The authentication was built using a set of 20 images. Any number of images can be used as your password or you can choose to use a normal text password.
